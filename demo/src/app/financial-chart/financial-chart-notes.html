This chart demonstrates how custom chart types can be used.

This custom chart type depends on a package called chartjs-chart-financial which is not yet on NPM.
It is included in the demo app as a git submodule directly from the github site of the package.

This also demonstrates using a different date/time plugin than moment .
This demo uses luxon instead, plugged into the chart.js module using the chartjs-adapter-luxon package.
